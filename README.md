## About TDD

TDD is way of ensuring that the code that we design does what we want out of it.

### Terms of TDD
* Units Tests 
	- Verify that individual piece of code works as expected.
* Features Tests
	- Written to test how the individual piece of code works together
* Regression Tests
	- Describe what an user is able to do 
* Http Tests
	- Test HTTP Requests (GET, POST, PATCH, PUT, DELETE)
* Authenticating Responses
	- Tests of Authentication & Authorization
* Database Tests
	- Tools & Assetions to test database driven applications
